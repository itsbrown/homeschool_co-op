Addressing the OpenAI API Quota Limit Issue
1. Issue Analysis
Error: "429 You exceeded your current quota, please check your plan and billing details."
Impact: Worksheet generation (/api/activities/generate) fails because the OpenAI API quota has been exceeded, preventing the creation of educational materials (e.g., worksheets for Tycoons class).
Current State:
The platform initially used Claude-3.7-Sonnet for generation but switched to OpenAI in recent updates (Sprint 5, Task 5.18), likely due to a misunderstanding or configuration change.
OpenAI’s 429 error indicates either:
The API key’s quota (e.g., requests per minute, total monthly usage) has been exceeded.
The account’s billing plan (e.g., free tier) has insufficient credits or limits for the workload.
Potential Causes:
Quota Limits: OpenAI’s free tier or low-tier plan might have strict limits (e.g., 20 requests/minute for free tier, or limited total tokens/month).
High Request Volume: Worksheet generation might be triggered frequently (e.g., by multiple users or background jobs), exceeding the quota.
Lack of Rate Limit Handling: The system lacks proper error handling for 429 errors, causing requests to fail without retry logic.
Switch to OpenAI: The recent switch from Claude-3.7-Sonnet to OpenAI (Sprint 5, Task 5.18) might have introduced this issue if OpenAI’s quota is lower than Claude’s.
2. Solution: Implement Robust Error Handling and Mitigate Quota Issues
Since the OpenAI API quota limit is the urgent issue, I’ll implement error handling for 429 errors, add retry logic, and explore alternatives to reduce dependency on OpenAI. Given the platform’s prior use of Claude-3.7-Sonnet, we can revert to Claude as a fallback if OpenAI quota issues persist.

Objective: Handle 429 errors gracefully, retry requests after a delay, and notify users of delays.

import time
import requests

def call_openai_api(data, retries=3):
    for i in range(retries):
        try:
            # Simulated OpenAI API call (replace with actual OpenAI call if used)
            response = requests.post('https://api.openai.com/v1/completions', 
                             headers={'Authorization': f'Bearer {os.getenv("OPENAI_API_KEY")}'},
                             json={'prompt': f"Generate a {data['activityType']} for {data['subject']}", 'max_tokens': 100})
            if response.status_code == 429:
                delay = 2 ** i  # Exponential backoff
                time.sleep(delay)
                continue
            response.raise_for_status()
            return response.json()
        except requests.exceptions.RequestException as e:
            if i == retries - 1:
                raise Exception(f"Failed to call OpenAI API after {retries} retries: {str(e)}")
    return None

const activityQueue = require('../queues/activityQueue');

router.post('/generate', auth(['admin', 'educator']), async (req, res) => {
  const { age, subject, activityType } = req.body;
  if (!age || !subject || !activityType) {
    return res.status(400).json({ message: 'Age, subject, and activity type are required' });
  }
  const job = await activityQueue.add({ age, subject, activityType });
  job.on('failed', (err) => {
    if (err.includes('429')) {
      res.json({ message: 'OpenAI API quota exceeded, retrying in background. You’ll receive an email when the activity is ready.' });
    } else {
      res.status(500).json({ message: 'Activity generation failed', error: err });
    }
  });
  job.on('completed', async (result) => {
    const activity = result;
    const doc = new PDFDocument();
    let buffers = [];
    doc.on('data', buffers.push.bind(buffers));
    doc.on('end', async () => {
      const pdfData = Buffer.concat(buffers);
      const key = `activities/${activityType}-${Date.now()}.pdf`;
      await s3.upload({
        Bucket: process.env.AWS_S3_BUCKET,
        Key: key,
        Body: pdfData,
        ContentType: 'application/pdf',
      }).promise();
      const downloadLink = s3.getSignedUrl('getObject', {
        Bucket: process.env.AWS_S3_BUCKET,
        Key: key,
        Expires: 3600,
      });
      const activityRecord = new Activity({
        type: activityType,
        url: downloadLink,
        subject,
        ageRange: age,
      });
      await activityRecord.save();
      await axios.post('http://localhost:3000/api/email/send', {
        email: 'user@example.com',
        subject: 'Activity Generated',
        message: `Your ${activityType} is ready: ${downloadLink}`,
      });
    });
    doc.fontSize(16).text(`${activityType} for ${subject}`, { align: 'center' });
    doc.moveDown();
    if (activityType == 'worksheet') {
      activity.questions.forEach(q => {
        doc.fontSize(12).text(`${q.question} (Answer: ${q.answer})`);
        doc.moveDown();
      });
    } else if (activityType == 'crossword') {
      doc.text('Crossword Puzzle');
      activity.clues.forEach(clue => doc.text(clue.across));
    }
    doc.end();
  });
  res.json({ message: 'Activity generation queued', jobId: job.id });
});
